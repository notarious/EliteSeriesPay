# AI-Assisted Development

This project uses Cursor AI as a development assistant.

Cursor is used for:

- planning small vertical features;
- reviewing architectural changes;
- checking layer boundaries;
- suggesting safe refactorings;
- generating repetitive boilerplate under project rules;
- identifying missing tests.

The project includes Cursor rules and commands to keep AI-assisted changes constrained by the architecture and business rules.

All generated code is manually reviewed before being committed.

AI assistance is treated as part of the engineering workflow, not as a replacement for design decisions, code review, or testing.
