# EliteSeriesPay

Приложение предназначено для локального учета участников, платежей и бюджета небольших платных сообществ.

Все данные хранятся только на вашем компьютере. Интернет требуется только для получения актуального курса валют при добавлении платежей в USD или EUR.

## Первый запуск

1. Запустите файл `EliteSeriesPay.jar`.
2. Через несколько секунд откроется браузер.
3. Если браузер не открылся автоматически, перейдите по адресу:

```
http://localhost:8080
```

При первом запуске приложение автоматически создаст локальную базу данных SQLite.

Все ваши данные будут храниться в файле:

```
eliteseriespay.db
```

рядом с файлом приложения.

## Резервные копии

Приложение автоматически создает резервные копии базы данных.

Также резервную копию можно создать вручную через раздел **«Резервные копии»**.

## Восстановление данных

Если основная база данных была повреждена или случайно удалена:

1. Закройте приложение.
2. Замените файл `eliteseriespay.db` одной из резервных копий из папки `backups`.
3. Запустите приложение снова.

## Очистка всех данных

Если нужно начать работу с чистой базой, используйте пункт меню **«Очистить все данные»**.

Перед очисткой приложение попросит подтвердить действие.

## Возможности

Приложение позволяет:

* создавать проекты;
* добавлять участников;
* вести несколько проектов одновременно;
* учитывать платежи в рублях, долларах и евро;
* автоматически рассчитывать комиссию VK Donut;
* получать текущий курс валют;
* отслеживать оплату абонементов;
* учитывать пакетных участников;
* просматривать историю платежей;
* получать отчеты по месяцам;
* рассчитывать количество оплаченных серий и остаток бюджета;
* создавать резервные копии базы данных.

## Обновление приложения

При выходе новой версии достаточно заменить файл приложения новой версией.

Файл `eliteseriespay.db` содержит все ваши данные и сохраняется отдельно, поэтому информация не потеряется.
