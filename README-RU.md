# Проекты от Ave Elite

Локальное приложение для учета участников, платежей и бюджета небольших платных сообществ.

Программа не требует установки отдельной базы данных или веб-сервера. Все данные хранятся только на вашем компьютере.

---

## Установка

1. Запустите файл **EliteSeriesPay-x.x.x.msi**.
2. Следуйте инструкциям мастера установки.
3. После завершения установки запустите программу через меню **Пуск** или ярлык на рабочем столе.

Через несколько секунд приложение автоматически откроется в браузере.

Если этого не произошло, откройте вручную:

```
http://localhost:8080
```

---

## Обновление

Для установки новой версии:

1. Закройте программу через меню значка в области уведомлений (**Выход**).
2. Запустите новый файл **EliteSeriesPay-x.x.x.msi**.
3. Следуйте инструкциям мастера установки.

Все пользовательские данные сохраняются автоматически.

База данных, резервные копии и журналы находятся в каталоге:

```
%LOCALAPPDATA%\EliteSeriesPay\
```

Если программа продолжает работать в фоновом режиме, установщик не сможет заменить ее файлы. Перед обновлением обязательно завершите работу приложения через меню значка в области уведомлений.

---

## Хранение данных

При первом запуске приложение автоматически создает локальную базу данных SQLite.

Все данные, настройки и резервные копии хранятся в профиле пользователя Windows и не передаются во внешние сервисы.

---

## Возможности

* управление несколькими проектами;
* учет участников проектов;
* поддержка абонементов и пакетных участников;
* регистрация платежей;
* автоматический расчет комиссии VK Donut;
* автоматическое получение курсов USD и EUR с возможностью ручной корректировки;
* история платежей участников;
* контроль просроченных оплат;
* отчеты по месяцам;
* расчет количества профинансированных серий;
* создание и восстановление резервных копий базы данных.

---

## Резервное копирование

Приложение автоматически создает резервные копии базы данных.

При необходимости резервную копию можно создать вручную через раздел **«Резервные копии»**.

---

## Сброс данных

Чтобы начать работу с пустой базой, воспользуйтесь пунктом меню **«Очистить данные»**.

При этом будут удалены все проекты, участники, платежи и настройки приложения.
