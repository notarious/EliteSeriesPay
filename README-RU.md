# EliteSeriesPay

EliteSeriesPay — программа для локального учета участников, платежей и бюджета небольших платных сообществ.

Все данные хранятся только на вашем компьютере.

## Установка

1. Запустите файл **EliteSeriesPay-x.x.x.msi**.
2. Следуйте инструкциям мастера установки.
3. После установки запустите программу через меню **Пуск** или ярлык на рабочем столе.

Через несколько секунд автоматически откроется браузер.

Если этого не произошло, откройте вручную:

```
http://localhost:8080
```

## Обновление

Чтобы установить новую версию поверх существующей:

1. Завершите работу программы через меню значка в области уведомлений: **Выход**
2. Запустите новый файл **EliteSeriesPay-x.x.x.msi**
3. Следуйте инструкциям мастера установки

База данных, резервные копии и журналы остаются в папке пользователя и не удаляются при обновлении:

```
%LOCALAPPDATA%\EliteSeriesPay\
```

Если программа всё ещё работает в фоне, установщик может не заменить файлы приложения. Перед обновлением обязательно выберите **Выход** в меню значка в трее.

## Где хранятся данные

При первом запуске программа автоматически создаст базу данных SQLite.

Все данные будут храниться в папке пользователя Windows.

Резервные копии также сохраняются автоматически.

## Основные возможности

Программа позволяет:

* вести несколько проектов;
* добавлять участников;
* учитывать абонементы и пакетных участников;
* регистрировать платежи;
* автоматически учитывать комиссию VK Donut;
* получать текущий курс USD и EUR;
* просматривать историю платежей;
* отслеживать просроченные абонементы;
* формировать отчеты по месяцам;
* рассчитывать количество оплаченных серий;
* создавать резервные копии базы данных.

## Резервные копии

Программа автоматически создает резервные копии базы данных.

Также резервную копию можно создать вручную через раздел **«Резервные копии»**.

## Очистка данных

Если необходимо начать работу заново, воспользуйтесь пунктом меню **«Очистить данные»**.

Будут удалены все проекты, участники, платежи и настройки.
